# Diabetes Predictor

This project predicts whether a person has diabetes based on inputs such as insulin levels, age, BMI, etc. Users need to provide these inputs, and the model will predict the likelihood of diabetes.

## Deployment

### Prerequisites

- pip
- Flask
- Anaconda
